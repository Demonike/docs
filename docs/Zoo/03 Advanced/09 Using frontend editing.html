Using frontend editing

Every user, with the 'edit' (or 'edit own') permission for the item, will automatically see an 'edit' link on the item page. The edit screen is customizable for each item type in the application manager. Additionally you can place the 'edit' element in every layout you like.

Custom apps must implement the changes introduced with ZOO 3.2. Click {doc: make-apps-ready-for-zoo32 text: here} to see how to make your App ready for ZOO 3.2.
