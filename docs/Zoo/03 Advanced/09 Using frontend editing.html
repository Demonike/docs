Using frontend editing

Every logged in user, with the 'edit' (or 'edit own') permission for the concering item, will automatically see an 'edit' link at the item page. The edit screen is customizable for every item at the application manager. Optional you can place the 'edit' element in every layout you like.

Custom apps must implement the changes introduced with ZOO 3.2. Click {doc: make-apps-ready-for-zoo32 text: here} to see how to make your App ready for ZOO 3.2.
