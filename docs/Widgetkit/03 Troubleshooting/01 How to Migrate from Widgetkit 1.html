How to Migrate from Widgetkit 1

Sorry, there is no upgrade path from Widgetkit 1.x to Widgetkit 2. This is because Widgetkit 2 has totally different widgets than Widgetkit 1.x. So it is impossible for us to provide any automatic upgrade.

Unfortunately, it is also NOT possible to run both Widgetkit 1.x and 2 at the same time. You have to decide which version you want to use. If you are satisfied with Widgetkit 1.x there is no need to move to the new version. If you start with a new site we recommend using Widgetkit 2.

To migrate your content from Widgetkit 1.x to 2, a possible solution is to get a full dump of your current website on your local system. There, you can remove Widgetkit 1, install the new version and recreate your widgets one by one. When you are done, you can update the online version with your prepared offline version. That way, you limit downtime of your live site to a minimum.

ZOO running with Widgetkit 1: Please note that as of right now, you will not be able to use Widgetkit 2 with ZOO. We are working on a solution for that.
