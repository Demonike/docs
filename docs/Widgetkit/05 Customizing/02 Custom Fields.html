Custom Fields

A neat Widgetkit 2 feature are the Custom Fields, simple yet powerfull allows Widgetkit to be used as a simple CCK.

Fields

When creating your New Content in the bottom you will see the Manage Fields button which will switch the view to the fields editing mode and allows you to create or manage the following fields.

Social Media

Field Description
Email Displays an Email To button.
Facebook Displays an Facebook button.
Google Plus Displays an Google+ button.
Twitter Displays an Twitter button.

Currently these fields are used by Grid, Grid-Stack, Map and Switcher Widget Plugins.
NOTE Social Media buttons will be displayed in the media overlay if set as Social Buttons.

Tags

Currently this field is not used by any Widget Plugin.

Location

Displays a Google Maps map. Currently is only used by the Map Widget Plugin.

Custom

This field is ment to be used with your custom Widget Plugins.
