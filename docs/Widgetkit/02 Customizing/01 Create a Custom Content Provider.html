Create a Custom Content Provider

Widgetkit is very modular and extendable by design. You can easily add your own content providers or widget renderer. Both, content provider and widget renderer, are plugins. You find them in the plugin folder of Widgetkit:

For Joomla
administrator/components/com_widgetkit/plugins/

For Wordpress
wp-content/plugins/widgetkit/plugins/

To create a new content provider or widget renderer just put it in the corresponding folder and it will be loaded automatically. If you are using a Warp 7 theme you can also put them in the theme folder.

For example
THEME-NAME/widgetkit/content/PROVIDER-NAME
THEME-NAME/widgetkit/widgets/WIDGET-NAME
