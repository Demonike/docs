Content Manager

In Widgetkit 2 content and widgets are decoupled, so that you can create them independently, widgets directly from the WordPress or Joomla editor and the content through the content manager.

When opening Widgetkit you'll see a New Content button with a select form that allows you to choose between creating custom content and loading content from Wordpress, Joomla, ZOO, K2 and Woocommerce.

Custom Content

When creating content for your widget, hit Add Item to add single content items. Alternatively you hit Add Media to select multiple items at once through the media library or manager of your CMS. Once you have selected an image, a preview will be visible below the media field. You can also type in the path directly. The WordPress media manager only allows you to access the upload folder. This is where being able to type your own the path comes in handy, so you can use a different folder structure.

In the Content field you can add text or Markup to be displayed as a caption or any other kind of content, depending on the widget renderer. You can also add a link to each content item. When configurating the widget, you will determine, how to display the link. For example, it can be a button or an image overlay.

Video

You can use videos as media items inside Widgetkit, no matter, if you want to embed YouTube or Vimeo videos or upload your own media. Additionally, you can add a poster image, which will be displayed on mobile devices.

Youtube and Vimeo

To embed a video in Widgetkit, just copy the video path from your video site into the media field. It is important that you use the path from the browser's address bar and not the embed code you will get when using the Share button. Otherwise the JavaScript from Widgetkit will not work correctly.

Additionally you can use some parameter to further control the video, for example to enable autoplay and loop. In YouTube you can also start the embedded video at a specific timestamp. Widgetkit will add its own API. These examples show the paths you would ideally use to implement a video from YouTube or Vimeo.

<!--This YouTube Video will start 100 seconds into the film, loop automatically, without controls or tone -->
http://www.youtube.com/watch?v=my-video-id&autoplay=1&loop=1&controls=0&showinfo=0&playlist=my-video-id&start=100

<!-- This Vimeo video will start and loop automatically, without controls or tone -->
http://vimeo.com/17808185?title=0&byline=0&autoplay=1&loop=1&setVolume=0

Upload videos

Of course you can also upload your own video files through the media manager of your CMS. In Joomla make sure that you enable uploading of video files in the Global Configuration.

Manage fields

Widgetkit allows you to add fields to the content manager. Just hit Manage fields, pick one of the field types from the select form and hit Close. The table below will give you an overview of the different field types.

Field Type Description
Email
Facebook
Google Plus
 Using these fields will create social buttons. Note that these will displayed in a media overlay, if Media Overlay is set to Social Buttons in the Content options of the widget manager.
Location With this field you can add a map to your widget content.
Tags With this field you can add a tags to your widget content.
Twitter With this field you can add a twitter account to your widget content.
Custom Here you can add custom fields. You can choose between Text, Textarea, Text, Boolean, Media and Location. This is useful, if you want to create a custom widget.
