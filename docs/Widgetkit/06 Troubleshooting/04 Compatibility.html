Compatibility

Editors

Sometimes, visual editors in your CMS will prevent the Widgetkit button to display. You can still insert and modify shortcodes manually, but you will not be able to open the visual editor. If you cannot see the Widgetkit button, try a different editor to see if that works. We are always working on making sure all major editors are supported.

Supported editors

There are no known issues with the following editors:

  • Plain textarea (no editor enabled)
  • JCE
  • RokPad
  • TinyMCE
  • CodeMirror
  • Ace

Unsupported editors

Known issues at the moment, but support is coming soon:

  • CKEditor
