Install the Widgetkit demo package

This tutorial is a step-by-step guide that will show you how to install a Widgetkit demo package.

Download the Widgetkit demo package

  1. Download the demo package from the {route: download-widgetkit text: Widgetkit downloads}.
  2. Extract the .zip file to a directory on your pc.
  3. Upload by FTP or copy the extracted directory to your webserver.
  4. Now just open your webservers url in your browser and follow the Joomla or WordPress installation instructions.

Troubleshooting (Joomla only)

On the last installation step don't forget to press the Install Sample Data button.
