Install Bonus Styles

Bonus styles are additional styles for widgets that are not part of the default Widgetkit release. These bonus styles are usually created for a theme to provide a nice integration with Widgetkit. To install a bonus style just follow these few steps.

  1. Download the latest {route: download-widgetkit text: Widgetkit bonus style package} and unpack all files to a directory on your pc.
  2. Every widget has its own folder with a /styles subdirectory containing all the additional styles.
  3. Pick a style and use FTP to upload a style to a the widgets styles directory on your webserver. For example a new Slideshow style would be uploaded to /media/widgetkit/widgets/slideshow/styles for Joomla and to /wp-content/plugins/widgetkit/widgets/slideshow/styles for WordPress.
  4. After you have copied the style you are able to select it as a "Style" option in your widget settings.
