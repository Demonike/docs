Widget Renderers

A widget renderer determines how your widget will be displayed in the frontend, for example as a Slideshow or a Grid. You can also add custom renderers. Each renderer accepts a set of configuration parameters. These can be set using the visual interface provided by Widgetkit or by changing the shortcode.

Grid

In a Grid widget, all items will be displayed as single items of a grid. Use the widget configuration to change the appearance of the grid.

Section Description
Columns Configure the number of columns for different screen sizes (for responsive behaviour). The gutter property determines the spacing between columns and rows.
Content Configure how each content item and its optionally attached media is rendered.
NOTE When setting Media overlay to Social Buttons, social network links that you have added through {doc:manage-fields text:Custom fields} when creating the content will be displayed in an overlay.
General Display or hide specific parts of your content. You can also set a custom HTML class that will be added to the widget element in the rendered output.

Example Shortcode

Default grid widget. Content items will have 100% each and will therefore be stacked.

[widgetkit id="23" widget="grid"]

A grid widget that displays 4 columns on large screens, automatically collapses to 2 columns on tablets and such and displays a single column on all devices smaller than that. To understand this mobile first approach, have a look at the UIkit responsive grid that is used here.

[widgetkit id="23" widget="grid" columns="1" columns_small="2" columns_large="4"]

Grid Stack

In a Grid stack, all items from your selected content will be rendered below each other. This can easily be used for a paired display of an image (taken from the media field of your content item) and a short text (taken from the content field of your content item).

Section Description
Grid Configure the appearance of the grid. The grid gutter determines the spacing between items in the grid.
Content Configure how each content item and its attached media is rendered. You can individually set animations for the media and content columns.
NOTE When setting Media overlay to Social Buttons, social network links that you have added through {doc:manage-fields text:Custom fields} when creating the content will be displayed in an overlay.
General Display or hide specific parts of your content. You can also set a custom HTML class that will be added to the widget element in the rendered output.

Example Shortcode

Default grid stack widget.

[widgetkit id="23" widget="grid-stack"]

A grid stack widget with dividers between the items, the media elements cropped to circles and animations that are shown when the item scrolls into view.

[widgetkit id="23" widget="grid-stack" divider=1 media_border="circle" animation_media="fade" animation_content="slide"]

Map

A map widget displays a Google map on your website. Make sure to add a custom field Location to your content element and select a location you want to display in the map. The other field of your content will be displayed when you click on the location marker. You can add several pins and popup boxes by adding more items to your content element. For more information, read the {doc:manage-fields text:Manage fields} doc.

Section Description
Map Configure the appearance and behaviour of the map.
Style Configure the styling of the map. You can remove all UI controls, invert the colours or set custom colours.
Popup Configure the popup that is shown at the location marker. You can individually set animations for the media and content columns.
NOTE When setting Media overlay to Social Buttons, social network links that you have added through {doc:manage-fields text:Custom fields} when creating the content will be displayed in an overlay.
General Display or hide specific parts of your content. You can also set a custom HTML class that will be added to the widget element in the rendered output.

Example Shortcode

A simple map widget. Make sure content (#23 in the example) has a custom field added for the location.

[widgetkit id="23" widget="map"]

A map showing the satellite image view and without any controlling UI elements.

[widgetkit id="23" widget="map" maptypeid="SATELLITE" disabledefaultui=1]

Slideshow

A slideshow turns your selected content element in a sequence of slides. Each item in your content element will represent one slide. Select a media element for an element as the slide's background. The title and content fields can be used as text captions on top of that image.

Section Description
Slideshow Configure the appearance and behaviour of the slideshow. Settings for the navigation and animations can be found here.
Caption When you select to enable the caption, a text will be displayed on each slide. The text is taken from title and content fields in your content item.
General Display or hide specific parts of your content. You can also set a custom HTML class that will be added to the widget element in the rendered output.

Example Shortcode

The default slideshow.

[widgetkit id="23" widget="slideshow"]

A slideshow with captions enabled and animations for slides and captions.

[widgetkit id="32" widget="slideshow" animation="swipe" caption=1 caption_animation="fade"]

Switcher

A switcher consists of a number of navigation toggles and their related content items. Switchers are useful for realizing little tabbed navigations and similar. Each item in your selected content element will result in a one navigation entry.

Section Description
Switcher Configure the appearance and behaviour of switcher. The navigation can be of different visual apperance and be positioned at any side of the content.
Content Configure how each content item and its optionally attached media is rendered.
NOTE When setting Media overlay to Social Buttons, social network links that you have added through {doc:manage-fields text:Custom fields} when creating the content will be displayed in an overlay.
General Display or hide specific parts of your content. You can also set a custom HTML class that will be added to the widget element in the rendered output.

Example Shortcode

The default switcher.

[widgetkit id="23" widget="switcher"]

A switcher with a tabbed navigation below the content.

[widgetkit id="23" widget="switcher" nav="tabs" position="bottom"]

A switcher with buttons for navigation displayed on the left.

[widgetkit id="23" widget="switcher" nav="nav" position="left"]
