UIkit

UIkit is YOOtheme's in-house frontend framework, on which all Warp 7 themes are built. For more information, take a look at the UIkit website.

UIkit documentation

UIkit has its own documentation, which you can also find on the UIkit website. You might want to take a particular look at the docs on Variables, Best theming practices and the customizer.json when starting out with Warp themes and UIkit.

UIkit in Warp

Folder/File Description
/less/theme.less Loads the UIkit theme uikit.less via the @import rule and contains the theme style.
/less/uikit Contains the UIkit theme files.
/less/uikit/uikit.less Loads the UIkit core uikit.less and all UIkit theme files via the @import rule.
/warp/vendor/uikit Contains the UIkit core files.
/warp/vendor/uikit/uikit.less Loads all UIkit core files via the @import rule.

Style variations of UIkit are saved in /styles/STYLE-NAME/style.less files. These files will be overridden by every saving process of the customizer in the theme settings.

Add your own UIkit theme

You can create your own UIkit theme by using the styles directory.

  1. Create a new /styles/STYLE-NAME/less/theme.less which will override the original file.
  2. Create a new /styles/STYLE-NAME/less/uikit directory and add your own styles/STYLE-NAME UIkit theme.
  3. Open the theme's config.php and register your own file instead of the theme.less: 
    'files' => array(
'/css/FILE-NAME.css' => 'less:FILE-NAME.less',
'/css/bootstrap.css' => 'less:bootstrap.less'
  4. Use the @import rule to load the /styles/STYLE-NAME/less/uikit/uikit.less file in the /styles/STYLE-NAME/less/theme.less. 
    @import "uikit/uikit.less";

// Your custom code here
  5. Use the @import rule to load the core uikit.less file in the /styles/STYLE-NAME/less/uikit/uikit.less. 
    // Default
@import "../../../warp/vendor/uikit/uikit.less";

// Theme
@...
  6. If needed, create a new /custom/less/customizer.json and add your custom variables.

NOTE Learn more about customization and overrides in the {doc: customization text: Customization} chapter.

UIkit and Bootstrap in Joomla 3

If you're using Joomla 3.x, you need to override the /less/bootstrap.less file to insert your style into Bootstrap. Warp offers a Bootstrap layer which converts the Joomla integrated Bootstrap styles into the UIkit theme styles.

  1. Copy the /less/bootrap.less and paste it in your /custom/less directory.
  2. Change the @import paths for the Joomla Bootstrap files, UIkit core and Bootstrap layer. 
    // Load Bootstrap Default
@import "../../../../media/jui/less/reset.less";
...

// Load UIkit Core
@import "../../warp/vendor/uikit/less/variables.less";

// Load Bootstrap Layer
@import "../../bootstrap/bootstrap-layer.less";
...
