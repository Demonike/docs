Feature and Compatibility Overview

Want to find out if a particular theme supports your CMS version and offers you all necessary features? Then please check the table below.

  • Theme: name of the theme
  • Warp Framework: Warp version on which the theme is based
  • CMS: supported version(s) of Joomla or WordPress
  • Responsive: availability of responsive layout that adapts to the viewing environment on mobiles, tablets and desktops
  • Mobile Theme: availability of dedicated mobile theme for iPhone, iPod touch and Android phones
  • RTL Support: availability of right-to-left language support
  • Demo Package: availability of demo package(s) for indicated CMS version(s)
